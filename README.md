# Alex Bachlej

I build Python backend tools, automation workflows, and small inspectable systems.

My current work focuses on practical developer tooling, reliability patterns, and controlled software delivery workflows. Most of my public repositories are recent, so I treat them as code samples rather than proof of long-term open-source activity.

## Public repositories

### production-engineering-samples

Reviewable Python samples for backend, security, and reliability patterns.

Includes:

- HMAC metadata signing and verification
- atomic filesystem writes
- safe upload handling
- cursor pagination helpers
- runtime drift records
- state transition validation
- lightweight audio timing maps

This is the best repository to review first.

### pixctl

Local image optimization and upscaling tool using Real-ESRGAN.

The project includes a local web UI and is intended as a practical desktop-style utility, not a framework.

### archon3

Early orchestration and planning prototype.

This is older experimental work and should be read as a prototype, not as a production system.

### blackbox-raven

Small local terminal/operator experiment for project-file injection, code generation, and session persistence.

This is a side experiment.
